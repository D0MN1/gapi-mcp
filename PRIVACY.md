# Privacy Policy

**gapi-mcp** is a personal, single-user MCP (Model Context Protocol) server that connects to Google Calendar and Google Tasks APIs.

## Data Collection

This application does not collect, store, or transmit any user data to third parties. All data stays on the user's local machine.

## Google API Usage

This application accesses Google Calendar and Google Tasks data solely on behalf of the authenticated user. OAuth credentials are stored locally and are never shared.

## Scopes

- `https://www.googleapis.com/auth/calendar` — Read/write access to Google Calendar
- `https://www.googleapis.com/auth/calendar.events` — Read/write access to calendar events
- `https://www.googleapis.com/auth/tasks` — Read/write access to Google Tasks
